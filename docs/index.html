Features

Cellular Automata first.

JCAL is implemented for building cellular automata easily. Complex Cellular Automata (CCA) are not supported yet.

Less is more.

Less code, less time waste. You can build your CA in a few code lines.

Designed for JAVA.

The library is written Java 17 for Java users.
